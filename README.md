# DiseasePrediction

To run this project:
- Clone the Project
- Open the Terminal and run the following commands
```
python3 -m pip install -r requirements.txt
```

```
streamlit run App.py  
```

- The Web-App should open directly but if it doesn't , click on the NETWORK URL.
